# 关于本站

欢迎来到我的博客！

这是一个使用 [Zola](https://www.getzola.org/) 静态网站生成器搭建的技术博客，主题采用自定义的 **x-theme**。

## 关于主题

x-theme 是一个具有 90 年代印刷美学的博客主题，采用醒目的橙色（#ff6b35）作为强调色，搭配硬阴影和粗边框设计，带有复古杂志风格。

## 技术栈

- **静态站点生成器**: Zola 0.22
- **模板引擎**: Tera
- **样式**: SCSS
- **部署**: GitHub Pages

## 关于我

一名热爱技术的开发者，专注于现代 Web 技术和开源项目。

如果你有任何问题或建议，欢迎通过 [Contact](/contact/) 页面与我联系。
