# About This Blog

Welcome to my blog!

This is a tech blog built with the [Zola](https://www.getzola.org/) static site generator, featuring a custom **x-theme**.

## About the Theme

x-theme is a blog theme with 90s print aesthetics, featuring a striking orange (#ff6b35) accent color, hard shadows, and bold borders with a vintage magazine style.

## Tech Stack

- **Static Site Generator**: Zola 0.22
- **Template Engine**: Tera
- **Styling**: SCSS
- **Deployment**: GitHub Pages

A developer passionate about technology, focused on modern web technologies and open source projects.
